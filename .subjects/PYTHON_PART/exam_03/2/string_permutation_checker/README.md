# string_permutation_checker

**Expected files:** `string_permutation_checker.py`

**Allowed functions:** Built-in Python functions

---

## Description

Write a function that checks if two strings are permutations of each other.
Whitespace should be ignored and the comparison is case-insensitive.

Your function must be declared as follows:

```python
def string_permutation_checker(str1: str, str2: str) -> bool:
```

### Examples

```
string_permutation_checker("listen", "silent")     => True
string_permutation_checker("hello", "world")       => False
string_permutation_checker("Triangle", "Integral") => True
string_permutation_checker("abc", "cab")           => True
```
