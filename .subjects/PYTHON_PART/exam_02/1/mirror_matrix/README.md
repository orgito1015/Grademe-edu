# mirror_matrix

**Expected files:** `mirror_matrix.py`

**Allowed functions:** All Python built-in functions

---

## Description

Write a function that mirrors a 2D matrix horizontally by reversing each row.

The function takes a 2D list of integers and returns a new 2D list where each row is reversed.

Your function must be declared as follows:

```python
def mirror_matrix(matrix: list[list[int]]) -> list[list[int]]:
```

## Example

```python
mirror_matrix([[1, 2, 3], [4, 5, 6]])  # Returns: [[3, 2, 1], [6, 5, 4]]
mirror_matrix([[1, 2], [3, 4], [5, 6]])  # Returns: [[2, 1], [4, 3], [6, 5]]
```
